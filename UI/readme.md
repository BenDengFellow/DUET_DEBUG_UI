创建了一个完整的PyQt6界面，包含4个标签页，窗口大小为1024x768像素。界面包含以下功能：

界面特点：
窗口大小：1024x768像素固定大小
标签页：4个标签页分别对应 Monitor、Functions、IO、Setting
各标签页功能：
Monitor标签页：

系统状态监控（CPU、内存、网络、设备连接状态）
系统日志显示区域
Functions标签页：

功能控制按钮（启动服务、停止服务、数据采集等）
参数设置（采样频率、工作模式）
IO标签页：

IO状态监控（数字输入/输出、模拟输入/输出状态）
IO控制功能（数字输出控制、模拟输出设置）
Setting标签页：

系统设置（语言、主题）
通信设置（串口波特率、网络端口）
保存/重置按钮
